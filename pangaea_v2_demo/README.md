# Pangaea v2 Demo
This directory contains demo scripts used at **2025 OCP Global Summit events**.
- [End-to-End Management of a CXL-enabled Datacenter Memory Fabric](https://www.opencompute.org/events/past-events/2025-ocp-global-summit)

Please check *0.nri_plugin.sh* script and *yaml/gigabyte-turin/\*.yaml* files for reference to use **_Pangaea v2_**.

Configurations are based on a Kubernetes cluster with CXL Fabric attached Worker Node named *"gigabyte-turin"*.

## Environment
Hardwares
- Gigabyte Turin Server
- VCS Enabled H3 Platform Falcon B2 CXL Switch
- Samsung CMM-D


Softwares
- CXL Agent
- DCMFM
    - Manager
    - Monitor
    - Agent
- CXL NRI Plugin
